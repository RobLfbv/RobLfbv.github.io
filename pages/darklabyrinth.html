The Dark Labyrinth

The Dark Labyrinth est un game concept que j'ai réalisé en binôme dans le cadre d'un cours sur le game design.
L'objectif était de faire un jeu avec la contrainte d'un support, ici, la support était hybride.

Pitch : "Enfuyez-vous d'un labyrinthe sur le point de s'effondrer. Déplacer votre pion pour découvrir le labyrinthe, remplir les objectifs et vous enfuir. Attention, le premier qui s'enfuit enferme les autres."
