Screen modifié venant d'un jeu

Ce travail est un montage fait sur Photoshop dans le cadre d'un cours d'infographie 2D.
L'objectif était de modifier un screenshot d'un jeu afin de changer l'univers dans lequel il se déroule.

Le jeu que j'ai choisi est Outer Wilds. A la place de faire dérouler le jeu dans un univers spatial, je me suis inspiré de "Vingt Mille lieues sous les mers", de l'imagerie Steampunk et de Bioshock pour modifier l'univers du jeu afin qu'il se déroule dans les fonds marin.
